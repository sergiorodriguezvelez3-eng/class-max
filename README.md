## Descripción
Class Max es una aplicación que permite a los estudiantes registrar materias, notas y calcular promedios académicos.

## Tecnologías
- Lenguaje: Python
- Base de Datos: SQLite
- IDE: Visual Studio Code
- Tipo de interfaz: Consola
